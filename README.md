### Demo of How to create and use playwright to write tests
[Demo Video](https://www.loom.com/share/ec1f0235e8fb4a3894faa40db81aa458)

To run tests without UI Mode
`pnpm run test`

To run tests in UI Mode
`pnpm run test:ui`

Start tracking the UI
`pnpm run codegen`

To show the test report
`pnpm run report`
